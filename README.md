# Multiplayer-SnakeGame
Real-time SnakeGame

### step-by-step

1. Use git clone
Use git clone (repo link)
Ex: `git clone https://github.com/heylucass/repo`

2. Use npm install
Ex: `npm install`
will install all dependencies
